Plug-in that displays book information from several post on one page.

== Description ==

Wordpress Book List allows to list all book reviews on my blog on one page in a big table. 

In the list are included 
- Author (boek_auteur)
- Title (boek_titel)
- Surname Author (boek_achternaam)
- Cover Image (boek_cover)

Fields can be made in Wordpress using Advanced Custom Fields.

CSS is customized for my own page, you can customize it for your own needs.

== Installation ==
1. Upload the plugin files to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3.  add a shortcode [custom_book_list] on your page.
